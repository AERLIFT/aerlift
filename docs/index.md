# AERLIFT Pipeline

Multi-instrument personal exposure monitoring pipeline built with Snakemake.
Ingests raw data from seven sensor platforms, standardizes to NetCDF4 (CF-1.8),
and prepares merged exposure datasets for analysis.

**Campaign:** AERLIFT — UC Berkeley School of Public Health
**PI:** Professor Ajay Pillarisetti, PhD, MPH
**Engineer:** Mark Campmier, PhD
**Lab:** BIRCHE³ Lab / Division of Environmental Health Sciences
